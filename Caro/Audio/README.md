# Audio - Âm thanh

Thư mục dành cho âm thanh và nhạc nền.

## Các thành phần dự kiến

- **Effects/** – Hiệu ứng âm thanh (click, đánh cờ, thắng/thua, menu...)
- **Music/** – Nhạc nền (menu, khi chơi)
- **Player/** – Module phát âm thanh (load, play, stop, volume)
- **Resources/** – File âm thanh (.wav, .mp3)

## Việc cần làm

- [ ] Chọn/cài thư viện phát âm thanh (ví dụ: Windows API, SDL_mixer, SFML...)
- [ ] Thu thập hoặc tạo file âm thanh phù hợp
- [ ] Viết module điều khiển phát nhạc và effect
- [ ] Tích hợp: play sound khi đánh cờ, thắng, thua, chọn menu
- [ ] Hỗ trợ chỉnh volume trong Settings
