# UI - Giao diện game

Thư mục dành cho giao diện người dùng (UI) giống các game thực tế.

## Các thành phần dự kiến

- **Screens/** – Các màn hình riêng biệt (Main menu, Game, Settings, Pause...)
- **Components/** – Các thành phần tái sử dụng (Button, TextBox, Cursor...)
- **Theme/** – Màu sắc, font chữ, ký tự vẽ khung
- **Assets/** – Art/Text dùng cho UI (logo ASCII, bảng, icon)
- **Layout/** – Vị trí, kích thước các vùng hiển thị

## Việc cần làm

- [ ] Thiết kế các màn hình (Main menu, Game screen, Settings, About...)
- [ ] Tạo thành phần UI dùng lại (button, khung, dialog)
- [ ] Định nghĩa theme (màu console, ký tự vẽ)
- [ ] Vẽ logo / banner ASCII cho game
- [ ] Ánh xạ layout rõ ràng cho từng màn hình
